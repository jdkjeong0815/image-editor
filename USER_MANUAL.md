# LEDCanvas Studio - 사용자 매뉴얼

**LEDCanvas Studio**는 대형 LED 전광판, 프로젝터 및 특수 디스플레이 환경에서 고해상도 이미지를 최상의 화질로 표출하기 위해 설계된 프리미엄 이미지 프로세서입니다. 현장 상황에 맞춘 즉각적인 색감 보정, 네온 효과, 그리고 자연스러운 시네마틱 애니메이션과 슬라이드쇼 기능을 제공하여 멈춰있는 이미지를 한 차원 높은 미디어 아트로 승화시키는 것이 목적입니다.

현재 버전을 기준으로 작성된 **LEDCanvas Studio**의 기능 및 버튼 설명 매뉴얼입니다.

---

## 0. 설치 및 실행 방법 (Windows)

본 프로그램은 웹 브라우저 기반으로 작동하지만, 보안 정책(CORS)으로 인해 반드시 **로컬 서버** 환경에서 실행해야 이미지들이 정상적으로 로드됩니다. 윈도우 노트북에서 실행하는 순서는 다음과 같습니다.

### 1단계: Python 설치 확인 및 설치 방법
* 대부분의 윈도우 환경에는 설치되어 있습니다. 확인을 위해 `Windows 키 + R`을 누르고 `cmd`를 입력하여 터미널을 엽니다.
* `python --version`을 입력했을 때 버전 정보가 나오면 준비 완료입니다.
* **만약 설치되어 있지 않다면:**
    1. **Microsoft Store(권장):** 윈도우 작업표시줄에서 'Store'를 검색해 연 뒤, 'Python'을 검색하여 최신 버전(예: 3.12)을 '다운로드' 하세요. 자동으로 경로 설정까지 완료됩니다.
    2. **공식 홈페이지:** [python.org/downloads](https://www.python.org/downloads/)에 접속해 노란색 Download 버튼을 눌러 설치 파일을 받으세요.
    3. **주의사항:** 공식 홈페이지 설치 파일 실행 시 반드시 하단의 **[Add Python to PATH]** 체크박스를 체크한 뒤 'Install Now'를 클릭해야 터미널에서 명령어가 인식됩니다.

### 2단계: 프로젝트 폴더로 이동
* 터미널(또는 PowerShell)에서 프로그램 파일들이 위치한 경로로 이동합니다.
  ```bash
  cd "I:\Antigravity\image-editor"
  ```

### 3단계: 로컬 서버 실행
* 아래 명령어를 입력하여 파이썬 내장 서버를 실행합니다.
  ```bash
  python -m http.server 8000
  ```
* 위 창을 닫지 않은 상태로 유지해야 프로그램이 작동합니다.

### 4단계: 웹 브라우저 접속
* 크롬(Chrome) 또는 엣지(Edge) 브라우저를 열고 주소창에 다음을 입력합니다.
  ```text
  http://localhost:8000
  ```

## 1. 상단 메인 메뉴

* **듀얼 모니터 버튼**
  * **기능:** 이미지를 띄울 수 있는 별도의 팝업창을 엽니다.
  * **설명:** 프로젝터나 두 번째 모니터에 전체 화면으로 이미지를 띄워놓고 컨트롤 패널(기존 창)에서 효과를 조작할 때 유용하게 쓰입니다. 본 화면에 적용된 모든 시각 효과가 듀얼 모니터 창에도 실시간으로 동기화됩니다.

* **저장하기 버튼**
  * **기능:** 현재 캔버스에 렌더링 된 이미지를 다운로드합니다.
  * **설명:** 밝기, 네온 효과 등 현재 적용된 모든 필터가 반영된 상태의 결과물을 `ledcanvas-edited.png` 파일로 로컬 PC에 저장합니다.

## 2. 이미지 효과 패널 (우측)

### 패널 상단 (빠른 동작)
* **LED 전광판 버튼**
  * **기능:** LED 디스플레이 송출에 최적화된 사전 설정(프리셋) 값을 즉시 적용합니다.
  * **설명:** 클릭 시 밝기 105%, 대비 130%, 채도 140%로 자동 조정되어, 다소 어둡게 보이는 LED 전광판 환경에서 이미지가 선명하고 쨍하게 보이도록 보정해 줍니다. 다시 누르면 원래 상태로 돌아갑니다.

* **초기화 버튼 (⟲ 아이콘)**
  * **기능:** 사용자가 조작한 모든 슬라이더 값을 기본 상태로 되돌립니다.
  * **설명:** 밝기/대비/채도 100%, 네온 강도 0% 등 원본 이미지 상태로 빠르게 복구할 때 사용합니다.

---

### 기본 조정

각 항목의 특징을 한 눈에 보기 쉽도록 색상과 함께 강조해 두었습니다.

* 💡 <span style="color:#FFA500; font-weight:bold;">밝기 (Brightness)</span>
  * **기능:** 이미지 전체의 명도를 시각적으로 조절합니다. (기본: 100%)
* 🌓 <span style="color:#4169E1; font-weight:bold;">대비 (Contrast)</span>
  * **기능:** 밝은 곳은 더 밝게, 어두운 곳은 더 어둡게 하여 이미지의 뚜렷함을 조절합니다. (기본: 100%)
* 🌈 <span style="color:#FF1493; font-weight:bold;">채도 (Saturation)</span>
  * **기능:** 이미지의 색감을 진하게 하거나 옅게(흑백에 가깝게) 조절합니다. (기본: 100%)

---

### 네온 효과 (Neon Glow)
* **밝기 임계값 (Threshold)**
  * **기능:** 네온 효과가 적용될 범위를 정하는 기준점입니다. (기본: 200)
  * **설명:** 숫자가 낮을수록 이미지의 조금 밝은 중간톤 영역까지 빛이 번지고, 숫자가 높을수록 아주 밝은 하이라이트 부분만 선택적으로 빛나게 됩니다.
* **네온 강도 (Intensity)**
  * **기능:** 선택된 영역이 발광하는 세기와 색상의 강렬함을 조절합니다. (기본: 0%)
  * **설명:** 강도를 올릴수록 빛이 더 강해지며 네온사인과 같은 쨍한 채도 증폭 효과가 덧씌워집니다.

---

### 시네마틱 경로 (Cinematic Animation)
카메라가 이미지 내부를 부드럽게 돌아다니는 역동적인 애니메이션 효과를 부여합니다.

* **포인트 지정 버튼 (0/4)**
  * **기능:** 애니메이션이 거쳐 갈 4개의 핵심 위치를 선택하는 모드를 켭니다.
  * **설명:** 버튼을 누르고 화면을 4번 클릭하면 순차적으로 포인트가 지정되며 선형으로 이어집니다.
* **애니메이션 재생 / 정지 버튼**
  * **기능:** 지정된 4개의 포인트를 따라 화면이 확대/이동하는 시네마틱 애니메이션을 시작하거나 멈춥니다. 
  * **설명:** 4개의 포인트를 모두 찍어야만 활성화됩니다.
* **속도 보정 슬라이더 (-5 ~ +5)**
  * **기능:** 카메라가 포인트와 포인트 사이를 한 번 이동하는 데 걸리는 **시간(초 단위 속도)**을 조절합니다.
  * **상세 설명:** 한 포인트를 향해 움직이는 **'이동 시간(Movement)'**과 도착 후 머무르는 **'정지 시간(Pause, 3초 고정)'**으로 구성되어 있습니다. 슬라이더의 값을 바꿀 때마다 이동에 걸리는 초단위 시간이 다음과 같이 변경됩니다.
    * **[ +5 ] 최대 속도:** 이동 75초 + 정지 3초 = 총 78초 소요
    * **[ +4 ]** : 이동 90초 + 정지 3초 = 총 93초
    * **[ +3 ]** : 이동 105초 + 정지 3초 = 총 108초
    * **[ +2 ]** : 이동 120초 + 정지 3초 = 총 123초
    * **[ +1 ]** : 이동 135초 + 정지 3초 = 총 138초
    * **[  0 ] 기본 속도:** 이동 150초 + 정지 3초 = 총 153초 소요
    * **[ -1 ]** : 이동 165초 + 정지 3초 = 총 168초
    * **[ -2 ]** : 이동 180초 + 정지 3초 = 총 183초
    * **[ -3 ]** : 이동 195초 + 정지 3초 = 총 198초
    * **[ -4 ]** : 이동 210초 + 정지 3초 = 총 213초
    * **[ -5 ] 최저 속도:** 이동 225초 + 정지 3초 = 총 228초 소요
  * **참고:** 숫자가 클수록(+쪽) 시간 단축(빠름), 작을수록(-쪽) 시간 연장(느림)입니다. 이동 시간 공식은 `(10 - 슬라이더 값) × 15초`를 따릅니다.

---

### 슬라이드쇼 설정 (Slideshow)
* **슬라이드 간격 슬라이더**
  * **기능:** 이미지 폴더 내의 사진들이 자동으로 넘어가는 대기 시간(간격)을 초 단위로 조절합니다.
  * **설명:** 최소 5초부터 최대 60초까지 설정 가능하며, 하단의 프로그레스 바가 다 차오르면 부드러운 교차 페이드 효과와 함께 다음 슬라이드로 넘어갑니다. (시네마틱 애니메이션 재생 중에는 동작을 일시 정지합니다.)
